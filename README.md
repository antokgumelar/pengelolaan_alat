# Pengelolaan Alat  
Sistem manajemen peminjaman dan pengembalian alat berbasis web, dikembangkan menggunakan PHP dan MySQL.  
Proyek ini ditujukan untuk membantu organisasi dalam mencatat katalog alat, memproses peminjaman dan pengembalian, serta menghasilkan laporan yang dibutuhkan.

## Fitur Utama  
- Manajemen katalog alat (tambah, edit, hapus alat)  
- Pencatatan data pegawai/peminjam  
- Proses peminjaman alat, proses pengembalian alat  
- Pencarian katalog dan pegawai  
- Laporan peminjaman & pengembalian  
- Interface berbeda untuk admin dan user biasa  

## Teknologi  
- Bahasa pemrograman: **PHP**  
- Database: MySQL  
- Framework: Bootstrap   
- Struktur folder utama:  
  - `index.php`, `login.php`, `home.php`, dll  
  - Folder `uploads/` untuk file upload
